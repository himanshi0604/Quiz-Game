Quiz App

A simple and responsive Quiz Application built using **HTML, CSS, and JavaScript**.  
This app includes a timer, progress bar, multiple-choice questions, and score tracking.

 Features

-  Countdown Timer for each question
-  Animated Time Progress Bar
-  Multiple Choice Questions
-  Instant Answer Feedback (Correct / Incorrect)
-  Question Number Tracking
-  Play Again Option
-  Responsive Design

 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)

 Project Structure

1. Click **Start Button**
2. Question appears with timer
3. Select an answer before time runs out
4. Move to next question automatically
5. Final score displayed at the end
